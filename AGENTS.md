# AGENTS.md

Instructions for OpenAI Codex and other OpenAI agents working in this repository.

Read [`CLAUDE.md`](CLAUDE.md) for repository structure, commands, architecture, and contribution guidance. The filename is retained for Claude Code compatibility; its instructions apply to all AI coding agents.

Read [`CONTEXT.md`](CONTEXT.md) for the project domain glossary before changing terminology, pipeline behavior, or architecture documentation.

Apply the same repository rules regardless of agent or model:

- Preserve existing user changes and avoid editing unrelated files.
- Keep credentials, tokens, and other sensitive values out of files and command output.
- Run the narrowest relevant lint or test checks after changes.
- Ask before destructive changes, security-model changes, or scope expansions.
